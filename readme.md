# Node express basic setup 

<ul>
<li>Clone project into your directory</li>
<li>Run 
```
 npm install

```
</li>
<li>
Create .env file
</li>
<li>Define following env variable in .env file
```
NODE_ENV
PORT
DATABASE_NAME
MONGO_URL
TRANSIT_SECRET
DB_SECRET
SENDGRID_API_USER
SENDGRID_API_KEY
EMAIL_FROM
API_BASE_URL
BASE_URL
ADMIN_BASE_URL
JWT_SECRET
TWILIO_ACCOUNTSID
TWILIO_AUTHTOKEN
TWILIO_FROM

```
</li>

<li>Finally run 
```
npm run dev
```
</li>

</ul>
